Readme for MyChat (Visual Studio 2012)

MyChat is a simple WPF application that demonstrates the usage of the AutoRefreshWrapper outlined in http://www.codeproject.com/Articles/233770/AutoRefresh-Entity-Framework-data-using-SQL-Server.

The program can be started several times on one or different computers. In each instance the user can enter a name and a message and then press the "Send" button. The message will be sent to the database and each instance will be notified about the change and will update the list of messages immediately.

Note that you might need to change the database connection by editing the application config (MyChat.exe.config). The default is ".\SQLEXPRESS" or "(localdb)\v11.0".

If the database (named "MyChat.Model.ChatContext" unless changed in MyChat.exe.config) doesn't exist, Code First will create it. Make sure that your database server is configured to work with Service Broker and that you're using sufficient credentials for login.
